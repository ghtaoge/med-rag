# 阿司匹林药品说明书

## 1. 药品名称

- 通用名：阿司匹林肠溶片
- 英文名：Aspirin Enteric-coated Tablets
- 商品名：拜阿司匹灵

## 2. 适应症

1. 降低急性心肌梗死疑似患者的发病风险
2. 预防心肌梗死复发
3. 降低短暂性脑缺血发作(TIA)及其继发脑卒中的风险
4. 预防脑卒中复发
5. 降低稳定性和不稳定性心绞痛患者的发病风险
6. 在冠状动脉搭桥手术(CABG)后预防血栓形成

## 3. 用法用量

- 成人常用量：口服，每日 100mg（1片）
- 建议在饭后用温水送服
- 不可掰开、压碎或咀嚼

## 4. 注意事项

1. 对阿司匹林过敏者禁用
2. 活动性消化道出血患者禁用
3. 严重肝功能损害者禁用
4. 孕妇慎用，尤其是妊娠晚期

## 5. 不良反应

| 反应类型 | 具体表现 |
| --- | --- |
| 胃肠道反应 | 恶心、呕吐、上腹部不适 |
| 出血风险 | 增加出血倾向 |
| 过敏反应 | 皮疹、哮喘 |

## 6. 药物相互作用

1. 与抗凝药合用增加出血风险
2. 与布洛芬合用可能降低阿司匹林的心血管保护作用
3. 与糖皮质激素合用增加胃肠道出血风险

# 布洛芬药品说明书

## 1. 药品名称

- 通用名：布洛芬缓释胶囊
- 英文名：Ibuprofen Sustained-release Capsules

## 2. 适应症

1. 缓解轻至中度疼痛：头痛、关节痛、偏头痛、牙痛、肌肉痛、神经痛、痛经
2. 普通感冒或流行性感冒引起的发热

## 3. 用法用量

- 成人常用量：口服，每次 300mg，每日 2 次
- 餐后服用可减少胃肠道不适
- 不宜长期使用，一般不超过 5 天

## 4. 注意事项

1. 对布洛芬过敏者禁用
2. 严重肾功能不全者禁用
3. 活动性消化道出血患者禁用
4. 心血管疾病患者慎用

## 5. 不良反应

| 反应类型 | 具体表现 |
| --- | --- |
| 胃肠道反应 | 恶心、胃灼热、腹泻 |
| 心血管 | 增加心肌梗死和脑卒中风险 |
| 肾脏 | 长期使用可能损害肾功能 |
